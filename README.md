# 🚗 Dashboard Interactivo — Mercado de Coches Usados en España

## Descripción del proyecto
Dashboard ejecutivo interactivo construido sobre datos reales de Milanuncios.com
para analizar el mercado de coches de segunda mano en España.

**Bootcamp:** Módulo II — Análisis y Visualización de Datos  
**Tipología:** Proyecto individual  
**Dataset:** Coches de segunda mano — Milanuncios (abril 2021)  
**Fuente:** https://zenodo.org/records/4674757  

---

## Problema de negocio
La dirección desconoce qué variables determinan el precio de venta,
qué comunidades concentran la oferta y qué tipo de vehículo genera
mayor interés — lo que impide optimizar el catálogo y la estrategia
de pricing.
